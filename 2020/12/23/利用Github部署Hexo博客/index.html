利用Github部署Hexo博客

2020-12-23 Hexo blog Comments

个人博客搭建步骤

  1. 搭建换件
  2. 初始化博客
  3. 写博客
  4. 部署博客
  5. 博客主题更换

环境搭建

  1. nodejs安装
    a.官网下载，并安装（Nodejs+npm）
    b.以管理员身份运行命令行
    c.cnpm替换npm加快安装速度：
     npm install -g cnpm --registry=https://registry.npm.taobao.org
 cnpm -v
    d.利用cnpm安装hexo
     cnpm install -g hexo-cli
 hexo -v

    博客初始化

  2. 新建一个blog文件夹
  3. 初始化博客：hexo init
  4. 启动/关闭Hexo：hexo s/ctrl+c
  5. 新建一个文章：hexo n “”
  6. 清理原有内容：hexo clean
  7. 重新生成：hexo g

    基本操作

  8. 创建修改博客
    注意：文件头冒号后面要加一个空格

  9. Markdown学习

  10. LaTeX公式

    部署到Github

  11. 新建仓库“用户名.github.io”
  12. 安装部署环境：cnpm install —save hexo-deployer-github
  13. 设置_config.yml deploy部分:
    a. type:git
    b. repo:第一步中新建仓库的地址
    c. branch：master
  14. 部署到Github：hexo d

    主题更换

  15. 下载主题到：themes/主题名文件夹
  16. 配置到博客：修改_config.yml中的theme，将其该问新下载的主题名
